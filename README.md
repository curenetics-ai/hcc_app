# Hepatocellular_Carcinoma_Response_Predictor

---

## What It Does

- Takes **patient data**:
  - Labs (AST, ALT, ALP, Albumin, Bilirubin, AFP)
  - Tumor staging
  - Demographics & comorbidities
  - Systemic & local treatments
  - Clinical notes (used for automated feature extraction)
- Predicts:
  - Likelihood of treatment success
  - Binary prediction (success/failure)
  - Toxicity probabilities per organ system
- Shows **feature contributions** (SHAP) for interpretability
- Allows **upload of outcomes** to track patient results over time
